# books-reviews
This is repository in which I have created a API which renders data to html templates from a JSON file. This is a book api with random books and their review.


# PROJECT SETUP 

- `fork repository`
- `clone repoitory`
- `npm install`
- `npm run dev`

# API ROUTES 

- `/api/books` OR `/api`
- `/api/book?id={bookID}`
